### Titluri

În `HTML` se pot utiliza 6 tipuri de titluri (headings) nivelul 1 fiind cel mai important
iar 6 cel mai puțin important:

```html
<h1> Titlu de nivel 1</h1>
<h2> Titlu de nivel 2</h2>
<h3> Titlu de nivel 3</h3>
<h4> Titlu de nivel 4</h4>
<h5> Titlu de nivel 5</h5>
<h6> Titlu de nivel 6</h6>

```

### Paragrafe

```html
<p>
    Textul paragrafului
    ...................
</p>
```

### Rând nou în paragraf

Limbajul `HTML` ignoră succesiunea de spații, tab-uri, caractere rând nou (New Line) dintr-un paragraf. Pentru trecerea forțată la rând nou se poate utiliza:

```html
<br>
```

### Text preformatat

Atunci când se dorește introducerea în document a unui text care să păstreze formatarea
cu spații, tab-uri, caractere rând nou se utilizează:

```html
<pre>
    Text preformatat
    ................
</pre>
```

### Linii orizontale

Trasarea unei linii orizontale pentru separarea logică a conținutului se face folosind:

```html
<hr>
```

### Formatări fizice pentru text

```html
<b>Text îngroșat (bold)</b>
<i>Text înclinat (italic)</i>
<u>Text subliniat (undeline)</u>
```

### Formatări logice pentru text

```html
<em>Text evidentiat</em>
<strong>Text puternic evidențiat</strong>
<del>Text eliminat</del>
<ins>Text adăugat</ins>
```

### Exponenți și indici

```html
<sup>exponent</sup>
<sub>indice</sub>
```
