Conectarea la serverul MySQL

January 21, 2022

Conectarea se face după un login cu succes la serverul Linux:

folosind comanda

mysql -u user -p

ca în exemplul de mai jos în care numele de utilizator este ariton:

Selectarea bazei de date de lucru

După conectare trebuie selectată baza de date de lucru al cărei nume este același cu numele de utilizator, folosind comanda:

use user_name
